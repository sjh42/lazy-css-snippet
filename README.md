# lazy-css-snippet

目前仅支持`scss`变量

## example

```json
{
  "9px": "$font-xxxs;", 
  "10px": "$font-xxs;",
  "11px": "$font-xs;",
  "12px": "$font-s;",
  "13px": "$font-m;",
  "14px": "$font-xm;",
  "16px": "$font-l;",
  "18px": "$font-xl;",
  "20px": "$font-xxl;",
  "24px": "$font-xxxl",
  "#1181d7": "$color-primary;",
  "#eda12d": "$color-yellow;",
  "#fdbb56": "$color-yellow1;",
  ...
}
```

以后可能会改为读取全部配置，不限文件类型的模式 (逃
